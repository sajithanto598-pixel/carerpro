🚀 CareerDev – Career & Placement Tracking Platform
CareerDev is a modern web-based platform designed to help students and professionals track their career growth, explore courses, and connect with opportunities.

🌟 Features
🎯 Career-focused homepage with modern UI
📚 Explore industry-relevant courses
🏫 Campus and location showcase
📊 Achievements and statistics tracking
🤝 Community engagement section
📩 Newsletter subscription
🧑‍💼 Free career consultation section
📱 Fully responsive design (mobile + desktop)
🍔 Hamburger menu for mobile navigation
🛠️ Tech Stack
HTML5
CSS3
JavaScript (Vanilla)
Responsive Design (Flexbox + Grid)
